# OnEdge HRIS System

A complete Human Resource Information System built with React, Node.js, and PostgreSQL.

## Features

- **Dashboard** — Real-time stats, charts, recent activity, upcoming birthdays
- **Employee Management** — Full CRUD, profiles, search & filter, CSV export
- **Organogram** — Interactive org chart with zoom controls
- **Leave Management** — Request, approve/reject, leave balances, calendar
- **Salary & Compensation** — Salary structures with allowances and deductions
- **Payroll** — Run payroll, generate items from salary structures, complete/cancel
- **Reports** — Headcount, leave analysis, payroll, salary — with charts & CSV export
- **Settings** — Departments, positions, leave types
